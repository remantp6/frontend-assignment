## How to run locally ?
* Clone or download the repo
  ```
  git clone https://github.com/remantp6/frontend-assignment
  ```
* Enter following command on the terminal
  ```
  npm start
  ```
  ## Folder structure
  ```
  |-src
  |-assets/
  |-components/
  |-pages/
  |-redux/
  |-services/
  ```
